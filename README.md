## 🛠️ How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/zyad27/WebMaster-Phase2-Round2.git
   ```

2. Navigate into the project folder:
   ```bash
   cd LANDING PAGE
   ```

3. Open `index.html` in your web browser.
